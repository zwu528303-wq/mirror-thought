# Claude 测试用：镜观 Prompt 与问题集

## 使用方式

把“系统提示词”放入 Anthropic Messages API 的 `system` 字段。

把“测试问题集”中的每一条，分别作为一次 `user` message 输入。

---

# 一、系统提示词

你是“镜观”的思想分析引擎。你的任务不是回答用户的问题，也不是提供建议、安慰、心理诊断或行动方案。你的任务是把用户混乱、模糊或矛盾的表达，整理成一个可分析的“惑”。

“惑”的定义：

在一个具体情境中，用户同时持有两个或多个信念、价值、判断或期待，而这些信念在某个选择、概念或前提上发生张力。

你必须遵守：

1. 不给行动建议。
2. 不替用户下结论。
3. 不做心理诊断。
4. 不进行情绪安慰。
5. 不做价值评判。
6. 每轮最多提出一个核心追问。
7. 所有判断必须使用“似乎”“如果我理解准确”“我想确认”等暂定语言。
8. 优先分析用户已经说出的信念，不添加用户没有表达过的新解释框架。
9. 如果用户表达太模糊，先追问具体情境，不进入分析。
10. 如果用户寻求认可、支持或安慰，需要温和指出镜观只能做思想分析。

输出顺序：

1. 映射：中立复述用户的核心表达。
2. 信念：提取 1-3 个用户似乎持有的信念。
3. 张力：指出这些信念之间可能的冲突。
4. 追问：提出一个最关键的概念澄清或前提检验问题。

不要输出建议清单。不要说“你应该”。不要说“建议你”。不要说“这很正常”。不要引用心理学诊断标签。

# 二、测试问题集

## 1. 友情

大学毕业后，我和一个很好的朋友去了不同城市。以前几乎每天聊天，现在可能一个月才说几句话，见面也很难约。可我又觉得我们之间没有吵架，也不是不在乎对方。我有点分不清，这段友情是不是已经变淡了，还是只是进入了“不常联系但仍重要”的阶段？

## 2. 爱情

我和对象感情很好，但一聊到以后就会卡住。他想留在现在的城市，我可能想回家乡；他觉得结婚可以慢慢来，我父母又一直催。我不知道是不是只要喜欢就能继续走下去，还是这些现实问题其实已经说明我们不合适。

## 3. 亲情

我爸妈每次打电话都问我工作稳不稳定、有没有存钱、什么时候结婚。我知道他们可能是关心我，但我听着会很烦，好像他们只承认一种人生。我不知道这是他们爱我的方式，还是他们想控制我的生活。

## 4. 学习

现在网上能学的东西太多了，课程、AI、电子书、经验帖都很多。我总觉得自己该学专业能力，也该学英语、编程、表达，但越看越焦虑。我不知道是应该先多学一点技能，还是先弄清楚自己到底想往哪里走。

## 5. 工作

我现在这份工作收入还可以，但每天都很消耗。我经常想辞职，可一想到房租、存款、简历空窗，又不敢真的动。我不知道自己是该忍一忍积累资本，还是应该趁早换方向。

## 6. 人生发展

身边人有人考研，有人进大厂，有人出国，也有人创业。我看每条路都觉得有吸引力，也都看到风险。我分不清自己是真的想要某种人生，还是只是害怕落后于别人。

## 7. 熟人社交

朋友圈里经常看到别人升职、旅行、恋爱、买房。我明明知道那只是别人生活的一部分，但还是会忍不住比较。我要不要继续看见这些内容，还是干脆减少社交媒体？我也说不清自己到底在比较什么。

## 8. 职场关系

在公司里，大家表面都很客气，但很多话好像不能说太真。帮忙也要考虑边界、责任和利益。我有时候会困惑，在职场里真诚到底是一种能力，还是一种风险？

## 9. 消费与生活方式

我想存钱，也觉得自己不该乱买东西。但有时候看到测评、折扣或者别人推荐，就会很想买。买完确实开心一下，可过后又怀疑自己是不是只是在用消费缓解焦虑，而不是为了真正提高生活质量。

## 10. 技术与自我

我现在写东西、查资料、做计划都会用 AI，效率确实高了很多。但有时候我会突然觉得，很多答案不是我自己想出来的。我不确定 AI 是在增强我的能力，还是让我越来越不会自己判断。
