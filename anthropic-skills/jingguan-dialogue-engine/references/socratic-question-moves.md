# Socratic Question Moves

Use one move per turn. Pick the move that clarifies the current "惑" with the least extra interpretation.

## 1. Definition

Use when the user relies on a key term without defining it.

Pattern:

```text
如果我理解准确，你这里说的“X”，更指 A，还是 B？
```

Examples:

- “真诚”更指表达真实想法，还是在利益关系中保持可信？
- “合适”更指感情状态，还是未来安排的可共同承担？
- “成功”更指资源位置，还是一种人生叙事被承认？

## 2. Reason

Use when a belief is clear but its ground is not.

Pattern:

```text
这个判断在你这里成立，主要依赖什么理由？
```

Examples:

- 你认为高频联系能代表友情重要性，这个判断主要来自过去经验，还是你对亲密关系的理解？
- 你觉得稳定收入是安全感来源，这里的安全感主要依赖现实成本，还是某种对未来的想象？

## 3. Relation

Use when two beliefs have been identified.

Pattern:

```text
这两个信念在你的处境里是互相支持，还是把你推向不同方向？
```

Examples:

- “陪伴家人很重要”和“个人发展很重要”在地点选择上如何相互拉扯？
- “AI 提高效率”和“独立判断重要”是在冲突，还是需要重新区分边界？

## 4. Fact / Value Split

Use when the user mixes what is true with what ought to be true.

Pattern:

```text
这里哪一部分更像事实判断，哪一部分更像你认为应当如此的价值判断？
```

Examples:

- “父母确实关心我”可能是事实判断；“关心不应当只等于稳定结婚存钱”可能是价值判断。
- “工作收入还可以”可能是事实判断；“生活不该被长期消耗”可能是价值判断。

## 5. Necessary / Negotiable Split

Use when the user is unsure whether a condition is a practical difficulty or a core premise.

Pattern:

```text
你说的 X 更像可以协调的差异，还是关系/选择成立的必要条件？
```

Examples:

- 城市、结婚节奏、父母期待中，哪一项更接近关系成立的前提？
- 工作消耗是阶段性成本，还是已经触及生活方式底线？

## 6. Hypothesis Elimination

Use when several interpretations are possible and the user has not distinguished them.

Pattern:

```text
你想分辨的更像 A，还是 B？
```

Examples:

- “怕落后”更像实际资源差距，还是人生叙事不成立？
- 消费带来的开心更像物品使用价值，还是购买瞬间的确定感？

## 7. Boundary Reminder

Use when the user asks for advice, reassurance, or validation.

Pattern:

```text
我不能替你判断该怎么做，但可以先帮你看清这个请求里同时存在的两个判断。你最想被确认的是事实、价值，还是关系中的责任？
```

Keep it short. Do not turn the boundary reminder into a lecture.

## 8. Vague Input Intake

Use when there is not enough material to form a "惑".

Pattern:

```text
现在的信息还不足以形成一个“惑”。你愿意先说一个最近让你卡住的具体情境吗？
```

Do not infer beliefs from vague emotion alone.
